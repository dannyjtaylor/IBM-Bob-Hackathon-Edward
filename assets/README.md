# Assets Directory

This directory contains icons and other resources for Edward.

## Required Assets

- **icon.png** - System tray icon (64x64 pixels recommended)
- **logo.png** - Application logo
- **sounds/** - Optional notification sounds

## Icon Guidelines

- Use the Edward color palette:
  - Gold: #DAA520
  - Red: #B22222
  - Silver: #A8A9AD
- Keep it simple and recognizable at small sizes
- PNG format with transparency

## Creating Icons

You can create custom icons using:
- Adobe Illustrator
- Figma
- GIMP
- Inkscape

Or use the default generated icon in the application.
